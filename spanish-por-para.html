🇪🇸 Por vs Para

Полный справочник с примерами и минимальными парами

PORпричина • обмен • через • время
1. Причина / мотив ▼
Trabajo por necesidad — Работаю из необходимости
Lo hice por ti — Я сделал это ради тебя
Gracias por todo — Спасибо за всё
Perdona por llegar tarde — Прости за опоздание
Estoy aquí por el trabajo — Я здесь из-за работы
2. Обмен / цена ▼
Compré el libro por 10 euros — Купил книгу за 10 евро
Cambié mi coche por uno nuevo — Обменял машину на новую
Te doy mi sandwich por tu postre — Меняю свой бутерброд на твой десерт
3. Движение через / вдоль ▼
Pasamos por Madrid — Мы проехали через Мадрид
Caminar por la playa — Гулять по пляжу
Viajar por Europa — Путешествовать по Европе
Entró por la ventana — Вошёл через окно
4. Время / длительность ▼
Viví allí por dos años — Жил там два года
por la mañana / tarde / noche — утром / днём / ночью
Estaré por aquí por una semana — Буду здесь неделю
5. Средство / способ ▼
por teléfono, por email, por internet
por avión, por tren, por barco
Mandé el paquete por correo — Отправил посылку почтой
6. Пассив и «за кого» ▼
El libro fue escrito por Cervantes — Книга написана Сервантесом
Hablo por todos nosotros — Говорю за всех нас
Voy por mi hermano — Иду вместо моего брата
por lo general, por ejemplo, por supuesto, por fin
PARAцель • для кого • направление • срок
1. Цель / назначение ▼
Estudio para aprender — Учусь, чтобы научиться
Un vaso para agua — Стакан для воды
¿Para qué sirve? — Для чего это?
Necesito dinero para el viaje — Нужны деньги для поездки
2. Получатель ▼
Este regalo es para ti — Этот подарок для тебя
Cocino para mi familia — Готовлю для семьи
Trabajo para una empresa grande — Работаю на большую компанию
3. Направление / пункт назначения ▼
Salgo para Madrid — Выезжаю в Мадрид
El tren para Barcelona — Поезд на Барселону
Vamos para casa — Идём домой
4. Срок / дедлайн ▼
Para el lunes — К понедельнику
Necesito esto para mañana — Нужно к завтрашнему дню
Para las cinco de la tarde — К пяти вечера
5. Сравнение / мнение ▼
Para mí, es fácil — По-моему, это легко
Para ser principiante, hablas bien — Для новичка говоришь хорошо
Es alto para su edad — Он высокий для своего возраста
6. Estar para (быть на грани) ▼
Estoy para salir — Я вот-вот выйду
No estoy para bromas — Мне не до шуток

Устойчивые выражения
POR
por favor por supuesto por ejemplo por fin por lo menos por lo general por lo tanto por cierto por eso por si acaso por lo visto por otra parte por casualidad
PARA
para siempre para nada para colmo para entonces no es para tanto para variar

Главное правило

POR = причина (почему?) • обмен/цена • движение через • длительность • средство • пассивный агент
PARA = цель (зачем?) • для кого • направление куда • к сроку • мнение • сравнение
Мнемоника: POR отвечает на «почему?» и «каким путём?» (причина, прошлое). PARA отвечает на «зачем?» и «для кого/куда?» (цель, будущее).
POR смотрит назад (причина). PARA смотрит вперёд (цель).

Estar por vs Estar para + инфинитив

  • estar por + inf. = ещё не сделано / быть за (поддерживать): Estoy por salir (Я за то, чтобы выйти); La casa está por limpiar (Дом ещё не убран)
  • estar para + inf. = вот-вот (готовность): Estoy para salir (Я вот-вот выйду); No estoy para bromas (Мне не до шуток)

Типичные ошибки русскоговорящих

Русское «для» не всегда = para. Если «для» означает причину/мотив, то нужен por:
Gracias para tu ayuda → Gracias por tu ayuda (Спасибо за помощь)
Русское «за» может быть и por, и para:
«Заплатил за книгу» = Pagué por el libro (обмен/цена)
«Деньги за (для) поездку» = dinero para el viaje (цель/назначение)
Длительность: в разговорном испанском por часто опускается:
Viví allí (por) dos años — оба варианта допустимы.
«Работаю на компанию» = Trabajo para una empresa (получатель/цель),
но «Борюсь за свободу» = Lucho por la libertad (причина/мотив).
